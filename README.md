# Classificadores Binarios 

Este projeto foi desenvolvido na disciplina de Inteligência Artificial e tem como objetivo construir um classificador binário capaz de identificar tráfego de rede normal ou malicioso (ataque) a partir de um conjunto de dados padronizado.

Para determinar qual algoritmo obtinha o melhor desempenho na tarefa de detecção, foram treinados e avaliados dois modelos supervisionados:

LightGBM — conhecido pela eficiência, velocidade e bom desempenho em dados tabulares.

Random Forest — modelo baseado em múltiplas árvores de decisão, robusto e eficaz para classificação binária.
